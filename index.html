Typography

Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

A paragraph can also be styled using the helper selectors like so:

this is a --primary paragraph

this is a --secondary paragraph

this is a --tertiary paragraph

this is a --error paragraph

this is a --warning paragraph

this is a --notice paragraph

this is a --success paragraph

this is a --basic paragraph

              
                <p class="--primary">this is a --primary paragraph</p>
                <p class="--secondary">this is a --secondary paragraph</p>
                <p class="--tertiary">this is a --tertiary paragraph</p>
                <p class="--error">this is a --error paragraph</p>
                <p class="--warning">this is a --warning paragraph</p>
                <p class="--notice">this is a --notice paragraph</p>
                <p class="--success">this is a --success paragraph</p>
                <p class="--basic">this is a --basic paragraph</p>

A paragraph can also use inherit the correct border color when paired with a color helper selector:

this is a --success --border paragraph

              
                <p class="--success --border">this is a --primary paragraph</p>

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.

Said no one, ever. 
              
                <blockquote>
                  <p>A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.</p>
                  <p>It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.</p>
                  <cite><a href="#">Said no one, ever.</a></cite>
                </blockquote>

Code blocks

Code blocks are highlighted with prism.js. to use prism you can do the write it the following way

The light code block with a border

              
                <pre class="S-Code-Block--shaded --red --border">
                  <code class="language-html">
                    <!-- code goes here -- >
                  </code>
                </pre>

The dark code block with a border

              
                <pre class="S-Code-Block--dark --red --border">
                  <code class="language-html">
                    <!-- code goes here -- >
                  </code>
                </pre>

The light code block without a border

              
                <pre class="S-Code-Block--dark">
                  <code class="language-html">
                    <!-- code goes here -- >
                  </code>
                </pre>

The regular code block with a without a border and with line-numbers

              
                <pre>
                  <code class="language-html line-numbers">
                    <!-- code goes here -- >
                  </code>
                </pre>

The regular code block without a border or line numbers

              
                <pre>
                  <code class="language-html">
                    <!-- code goes here -- >
                  </code>
                </pre>
